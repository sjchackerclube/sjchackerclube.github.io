Um site em 5 minutos. Bottle.py

Aprender desenvolvimento WEB pode não ser a tarefa mais fácil na área de programação. A primeira barreira a ser quebrada é montar um ambiente complicado, digno de competir com um amazon web server, só pra exibir um “Hello World” no browser. Mas será que tem um jeito mais fácil de quebrar esta barreira e explorar o crescente mundo do desenvolvimento web? Sim, a resposta é o Bottle.py. Um framework WEB minimalista (porem não menos competente) que torna possível ao programador subir um servidor web em 5 linhas, podendo assim se preocupar com o que realmente interessa: código. Nesta palestra pretendo apresentar alguns conceitos básicos de programação web como: mapeamento de url, request handlers, métodos de envio, templates e como o MVC ajuda no desenvolvimento WEB.
