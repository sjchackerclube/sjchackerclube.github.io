Palestra

HHVM, XHP and Hack - Build your own Facebook

Facebook's frontend is based on three main pillars: HHVM, the HipHop Virtual Machine, that compiles PHP/Hack to bytecode and then JIT compiles it to machine code at runtime; XHP, a PHP extension that transforms HTML in valid PHP expressions and objects; and Hack, a statically typed PHP with added features, like generics, lambdas, etc. The goal of this talk is to introduce those concepts and give some concrete but simple examples.
